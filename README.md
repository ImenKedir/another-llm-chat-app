# Naughty ML

Generative AI SaaS

## Run Locally

Clone the project

```bash
https://github.com/ImenKedir/naughtyml
```

Go to the project directory

```bash
cd naughtyml/remix
```

Install dependencies

```bash
pnpm install
```

Start the sst development server

```bash
pnpm sst dev
```

Start the remix development server

```bash
pnpm run dev
```
